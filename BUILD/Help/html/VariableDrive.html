Variable drive letters


USB sticks often get different volume names assigned when plugged into different computers. FreeFileSync offers two solutions to handle this issue:


Option 1: Specify a directory via volume name: Syntax: [<volume name>]\<path>

Use "[ZenJu-USB]\directory" instead of "E:\directory" where "ZenJu-USB" is the name of the USB stick.



It is not required to look up and enter the volume name manually. Just select the corresponding entry in the drop down menu.


Option 2: Use a relative directory name:

  • Use "\directory" instead of "E:\directory"
     

  • Save and copy synchronization settings to the USB stick: "E:\settings.ffs_gui"
     

  • Start FreeFileSync by double-clicking on "E:\settings.ffs_gui"
    → Working directory is automatically set to "    E:\" by the operating system so that "\directory" will be resolved as "E:\directory" during synchronization.
