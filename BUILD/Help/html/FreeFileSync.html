FreeFileSync
- Folder Comparison and Synchronization -

Usage:

  1. Choose left and right directories.

     

  2. "Compare" them.

     

  3. Select synchronization settings.

     

  4. Press "Synchronize..." to begin synchronization.



Main Dialog:


  1. Start comparison

  2. Specify comparison settings

  3. Set sync configuration

  4. Start synchronization

  5. Add additional folder pairs

  6. Swap sides

  7. Select left and right directories

  8. Save/load configuration

  9. Filter files

  10. Specify category for display

  11. Synchronization statistics/preview



Key Features

  1. Detect moved and renamed files and folders

  2. Copy locked files (Volume Shadow Copy Service)

  3. Detect conflicts and propagate deletions

  4. Binary comparison

  5. Full support for Symbolic Links

  6. Run as a batch job

  7. Multiple folder pairs

  8. Preserve NTFS sparse files

  9. Support very long path names > 260 characters

  10. Transactional file copy

  11. Cross-platform: Windows/Linux

  12. Expand environment variables like %USERPROFILE%

  13. Access variable drive letters by volume name (USB sticks)

  14. Native 32 and 64-bit builds

  15. Versioning of deleted files

  16. Optimal sync sequence prevents disc space bottlenecks

  17. Full unicode support

  18. Highly optimized performance

  19. Include/exclude files via filter

  20. Local or portable installation

  21. Handle daylight saving time changes on FAT/FAT32

  22. Use macros %time%, %date%, et al. for recurring backups

  23. Case sensitive synchronization

  24. Built-in locking serializes multiple jobs running against the same network share
