FreeFileSync
- Folder Comparison and Synchronization -

Usage:

  1. Choose left and right directories.

     

  2. "Compare" them.

     

  3. Select synchronization settings.

     

  4. Press "Synchronize..." to begin synchronization.



Main Dialog:


  1. Start comparison

  2. Specify comparison settings

  3. Set sync configuration

  4. Start synchronization

  5. Add additional folder pairs

  6. Swap sides

  7. Select left and right directories

  8. Save/load configuration

  9. Exclude files

  10. Specify category for display

  11. Synchronization statistics/preview



Menu bar:


Menu "File"

  • 1. Compare
    Start comparing both sides according to comparison settings (compare by filesize and date or by content).

  • 2. Synchronize...
    Begin synchronization and copy files as seen on the middle grid of the main dialog.

  • Switch View
    After comparison has finished the relation between files on left and right side can be visualized by two views:

1. „Comparison Result“ shows a category for each row like „existing on left side only“ or „left file is newer“.

2. „Synchronization Preview“ is the result of the synchronization configuration applied on each category. The effective directions in which the files will be copied are shown.

  • New
    Clears all configuration data and restores an initial state.

  • Save Configuration...
    Save the current settings into a file.

  • Load Configuration...
    Load configuration from file.

  • Quit
    Exit FreeFileSync.


Menu "Advanced"

  • Language
    Change the language of the tool.

  • Global Settings...
    A number of settings that have a global status in the sense that they will be applied to all GUI and batch configurations (*.ffs_gui and *.ffs_batch).

  • Create Batch Job
    Creates a FreeFileSync batch file (*.ffs_batch) that executes a predefined synchronization without user intervention.

  • Export File List...
    Export all data that     is currently shown on the main grids as a *.csv file that can be imported into Excel.


Menu "Help"

  • Content
    Show this helpfile.

  • Check for new version
    Check online whether a more recent version of FreeFileSync is existing.
